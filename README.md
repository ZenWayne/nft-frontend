# nft-frontend
NFT frontend for [i6o6i/ERC721](https://github.com/i6o6i/erc721)

Powered By React, Web3js, Helia

[DEMO](https://nft.kuanzw.com/)

### Prerequisite
npm

### Build

1. install npm dependencies
``` shell
npm install
```

2. build
``` shell
npm build
```

3. run for development
use react-script to run
```shell
npm start
```
